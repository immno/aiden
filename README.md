## Project Setup

```sh
npm install
cargo install tauri-cli --version "^2.0.0"
```

### Compile and Hot-Reload for Development

```sh
cargo tauri dev
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
